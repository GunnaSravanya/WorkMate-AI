# Telugu Query Examples - Test Guide

## Now the chatbot understands these Telugu queries:

### Eligibility Questions (అర్హత)
- "నాకు రుణం అర్హత ఉందా?" (Am I eligible for a loan?)
- "నేను లోన్ పొందవచ్చా?" (Can I get a loan?)
- "రుణం కోసం దరఖాస్తు చేయవచ్చా?" (Can I apply for a loan?)
- "క్రెడిట్ ఆమోదం ఎలా దొరుకుతుంది?" (How to get credit approval?)

### Scheme Questions (పథకాలు)
- "ప్రభుత్వ పథకాలు ఏమిటి?" (What are government schemes?)
- "నాకు ఏ యోజన అందుబాటులో ఉంది?" (Which scheme is available for me?)
- "స్కీమ్ ప్రయోజనాలు ఏమిటి?" (What are scheme benefits?)
- "ప్రభుత్వ సౌకర్యం గురించి చెప్పండి" (Tell me about government facilities)

### Document Questions (పత్రాలు)
- "ఏ పత్రాలు అవసరం?" (What documents are required?)
- "డాక్యుమెంట్స్ కావాలి?" (Need documents?)
- "సర్టిఫికేట్ ప్రూఫ్ ఏమిటి?" (What certificate proof?)
- "తప్పనిసరి పత్రాలు ఏవి?" (What are mandatory documents?)

## How It Works Now

The chatbot now has **comprehensive keyword detection** for:
- ✅ **Telugu words**: అర్హత, రుణం, పథకం, పత్రాలు, etc.
- ✅ **Hindi words**: पात्रता, ऋण, योजना, दस्तावेज, etc.
- ✅ **English words**: eligible, loan, scheme, document, etc.

## Test It
1. Run `npm run dev`
2. Switch to Telugu (తెలుగు)
3. Type or speak any of the above queries
4. The bot should now give the correct response!
